## Preparación del Entorno

### Linux
```bash

python3 -m venv .venv

source .venv/bin/activate

```

```bash
python -m venv .venv

.\.venv\Scripts\activate

pip install -r requirements.txt
```

```bash
source .venv/bin/activate

```

### Windows
